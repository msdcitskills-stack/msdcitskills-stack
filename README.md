# 👋 Welcome

Welcome to the **School of IT Skills, MSDC (Manipal Skills Development Centre)** GitHub!

We are dedicated to empowering students, professionals, and aspiring developers with **industry-focused IT education** through practical learning, hands-on projects, and modern technologies.

Our mission is to bridge the gap between **academics and industry** by providing real-world technical skills.
